## 💇🏻‍♂️ Sobre o projeto
Bem-vindo ao repositório do projeto de criação de tarefas em React Native!
Este projeto foi desenvolvido para fornecer uma aplicação simples e eficiente
para a gestão de tarefas em dispositivos móveis, utilizando a tecnologia React Native.

# Algumas das principais funcionalidades do aplicativo:

Criação de Tarefas: Adicione suas tarefas facilmente com apenas alguns toques.
Edição e Exclusão: Atualize ou remova tarefas concluídas com simplicidade.
Design Intuitivo: Interface amigável e intuitiva para uma experiência de usuário agradável.
Armazenamento Local: As tarefas são salvas localmente no dispositivo para acesso rápido e offline.

## 💻 Começando

### Requirements

Ambiente configurado para emular aplicativos.

**Clone o projeto e acesse a pasta**

```bash
$ git clone git@github.com:CristianoVieira1/teste-developer.git && cd teste-developer
```

**Siga os passos abaixo**

```bash
# Install the dependencies
$ yarn

# If you are going to emulate with android, run this command
# Be sure to have the emulator open
$ yarn android

# If you are going to emulate with ios, run this command
$ yarn ios
```

## 🤔 como contribuir

**Faça uma bifurcação deste repositório**

```bash
# Fork using GitHub official command line
# If you don't have the GitHub CLI, use the web site to do that.

$ gh repo fork CristianoVieira1/teste-developer
```

**Siga os passos abaixo**

```bash
# Clone your fork
$ git clone your-fork-url && cd teste-developer

# Create a branch with your feature
$ git checkout -b my-feature

# Make the commit with your changes
$ git commit -m 'feat: My new feature'

# Send the code to your remote branch
$ git push origin my-feature
```

Depois que sua solicitação pull for mesclada, você poderá excluir sua ramificação
